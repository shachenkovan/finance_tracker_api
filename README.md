Система управления личными финансами с бюджетированием и целями накопления.

Ключевые функции:
- Регистрация/аутентификация
- Управление кошельками и транзакциями
- Бюджетирование по категориям
- Отслеживание финансовых целей
- Аналитика и отчеты

Стек:
- Python
- FastAPI
- SQLAlchemy
- Pydantic
- PostgreSQL
- Alembic (в разработке)
- JWT
- Docker
- Docker Compose
- GitHub Actions (в разработке)
- pytest
- Redis (в разработке)


Что может делать пользователь при помощи API:
- Создать кошелек ("Наличные", "Карта Тинькофф")
- Добавить категории ("На подарки", "На новый год", "Зарплата")
- Вносить операции (+5000 зарплата, -300 обед)
- Ставить цели ("Накопить на iPhone к декабрю")
- Планировать бюджет ("В этом месяце на еду 15000р")
- Смотреть аналитику ("В этом месяце потратил на еду 5000р")
